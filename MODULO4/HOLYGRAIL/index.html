GRID-TEMPLATE-AREAS

El layout "Holy Grail" se implementa de forma elegante y semántica utilizando grid-template-areas. Cada sección principal de la página (header, nav, main, sidebar, footer) es un "área" nombrada en el contenedor principal.

Esta técnica permite una reestructuración completa para vistas móviles simplemente redefiniendo las áreas dentro de una media query, lo que proporciona un control declarativo y muy legible sobre el layout en diferentes breakpoints.

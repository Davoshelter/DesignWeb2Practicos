Laboratorio de NavBars Pegajosas

Una colección de diferentes implementaciones de position: sticky.

1. Sticky Básico

Este es el ejemplo más simple. La barra de navegación se desplaza con la página hasta que llega a la parte superior, donde se queda "pegada". No requiere JavaScript, solo position: sticky; top: 0;.

Simplicidad

Fácil de implementar y muy robusto.

Rendimiento

Nativo de CSS, no sobrecarga el navegador.

Compatibilidad

Soportado por todos los navegadores modernos.

2. Sticky con Cambio de Estilo

Esta barra empieza siendo semi-transparente. Al hacer scroll, una clase .scrolled se añade vía JavaScript, cambiando su fondo a un color sólido y añadiendo una sombra. Muy útil sobre cabeceras con imágenes.

Dinamismo

Crea una transición visual clara.

Legibilidad

Asegura que el texto del navbar se lea bien.

Contextual

Se adapta al contenido que está debajo.

3. Sticky que se Encoge (Shrink) Logo

Al hacer scroll, esta barra reduce su altura (padding) y el tamaño de la fuente. Es una técnica elegante para maximizar el espacio visible del contenido. También se controla con una clase .scrolled desde JavaScript.

Eficiencia

Menos espacio de navbar, más para el contenido.

Elegancia

Una micro-interacción que denota calidad.

Foco

El contenido principal gana protagonismo.

4. Sticky que Aparece al Subir

Esta barra se oculta cuando haces scroll hacia abajo para no estorbar, pero reaparece inmediatamente en cuanto empiezas a subir. Requiere JavaScript para detectar la dirección del scroll.

Inmersivo

La interfaz se quita del medio al leer.

Intuitivo

Reaparece cuando el usuario muestra intención de navegar.

Avanzado

Una técnica de UX moderna y sofisticada.
